# f42v — herbal (120 words)

TRANSLATION:

Take the plant and filter it with soft cotton material, anoint with clear bright liquid. Pour the preparation, pour it strong, filter with clear bright liquid. Apply the filtered preparation, let it rest with cotton material - it will be bitter at first but sweet when completed. 

Make the mixture strong from what is filtered, let it set properly. Pour it drop by drop carefully into bright liquid. Apply this filtered preparation by pouring into a vessel with chalk to clarify it. Set it to rest and pour the filtered preparation.

In bright liquid, apply this preparation by pouring - it should rest and be poured again. Anoint the mixture, filter it with cotton, apply bright liquid in drops. Filter the preparation, let the mixture rest with bright liquid in strong solution. Apply bright filtered preparation by pouring, rest it in bright liquid.

Filter the mixture and set it, apply by pouring in bright clear liquid, it should be filtered bright. Filter the strong mixture, apply by pouring carefully, dripping sweet liquid. Pour with cotton and set it.

Set it and filter it out.
