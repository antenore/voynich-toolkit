# f8r — herbal (152 words)

TRANSLATION:

For anointing and applications, take the plant material and anoint. Rest and apply medicine to the preparations. Extract the juice from the filtered material using strong drink as a clear solvent. Rest the preparations until bright and clear.

Anoint and pour the mixture repeatedly. Add chalk to clarify, then pour the strong drink solvent. Set the preparation and let it become bright and clear.

Pour the filtered extract. Apply medicine and anoint the clear preparation.

Within the clear solvent, pour and filter repeatedly until bright. Apply the clear preparation to affected areas using bitter indicators to test strength.

Apply the bright, clear preparation using strong alcoholic extraction until the mixture becomes properly bright.

The completed preparation:

Take the extract and apply to affected areas. Pour and set with rose preparations using sweet-testing mixtures. 

Apply the completed cotton filtering with strong drink solution. Set the mixture until bright, testing with bitter indicators. Filter through cotton material using sweet completion tests.

Apply strong drink solution within the affected areas using cotton applications until bright.

Apply to the affected plant areas within bright preparations using filtering applications.

Remove by filtering the plant preparations within the affected areas repeatedly.

Remove and filter properly.

Apply the completed filtering preparations.
